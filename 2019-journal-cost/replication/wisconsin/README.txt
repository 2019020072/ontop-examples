Content of this folder:

- data: Contains the data dump and data schema.
- wisconsin6.sparql: Input SPARQL Queries
- wisconsin6.jucq.sparql: Cover SPARQL Queries. In the file, f_1 means "First fragment", f_2 means "Second fragment".
- wisconsin6.obda: Mappings file
- unfoldings: UCQ and JUCQ unfoldings for the SPARQL queries and cover queries, respectively.
- cost-estimation: scripts for estimating the costs of the queries
